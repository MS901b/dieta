Mapa Introdução Início

Para encontrar o ponto que indica a dieta mais barata, o primeiro passo é identificar quais são os vértices do polígono obtido na parte anterior.

Veja que o vértice V1 é dado pela intersecção de duas retas. Para encontrá-lo, basta achar essa intersecção entre elas, ou seja, a solução de um sistema linear 2x2 formado pelas equações de cada uma das retas.
